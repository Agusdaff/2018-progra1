Geriatrico
============

1- Definir una estructura de datos que permita almacenar los datos de los geriatricos
2- Mostrar un menu principal que contenga las siguientes opciones:
	a- Importar datos
	b- Alta Nuevo Geriatrico 
	c- Búsqueda Geriatricos
	d- Exportar Geriatricos por Barrio
	e- Estadisticas de Geriatricos por Barrio
	z- Salir

3- Importar Datos: el sistema deberá importar todos los datos del archivo de texto "geriatricos.txt" y cargarlo en un vector de estructuras en memoria, reemplazando todos los datos existentes

4- Alta Nuevo Geriatrico: el sistema solicitará al usuario los datos del nuevo geriatrico y lo agregará en el vector en memoria en una posición libre

5- La búsqueda de geriatricos se realizará de la siguiente forma:
	a- Solicitar al usuario el campo por el que desea buscar: 
		1- Nombre
		2- Contacto
	b- Realizar la búsqueda por el campo seleccionado con criterio "contiene", ejemplo: si el usuario ingresa "HOGAR" debe encontrar, entre otros, EL HOGAR DE DORA y HOGAR SAN NICOLAS  
	c- Mostrar la lista de geriatricos encontrados en páginas (solamente las columnas ID, NOMBRE y CONTACTO), haciendo una pausa cada 20 registros, donde el usuario ingresará "C" para continuar "V" para visualizar o "E" para editar. Si selecciona visualizar o editar, se le debe pedir el ID del registro a visualizar/editar.

	d- Si seleccionó continuar mostrar la siguiente página de la lista de la misma forma que en el punto c

	e- Si seleccionó editar mostrar el registro seleccionado y luego pedir campo a campo todos los datos, dando al usuario, de alguna forma, la opción de no modificarlo y que se mantengan los datos que el registro ya tiene. Al terminar de editar vuelve al menu principal.

	f- Si seleccionó visualizar deberá mostrar todos los datos del registro seleccionado, al presionar una tecla para continuar vuelve al menu principal.

6- Exportar Geriatricos por Barrio debe solicitar al usuario un barrio y en base al barrio ingresado guardar en un archivo de texto todos los geriatricos del barrio seleccionado	

7- La estadística de geriatricos por barrio deberá mostrar por pantalla una tabla de dos columnas, en la primera el nombre del barrio y en la segunda la cantidad de geriatricos que hay en dicho barrio.

8- El sistema antes de salir deberá guardar los datos del vector en memoria en un archivo binario

9- El sistema al abrir deberá cargar del archivo binario los datos del vector en memoria
